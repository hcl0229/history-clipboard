# scripts/ — 工具脚本

> 版本：v1.0 | 日期：2026-06-09

此目录存放项目相关的构建、迁移、开发辅助脚本。

## 目录约定

| 子目录/文件 | 用途 | 说明 |
|------------|------|------|
| `migrate/` | 数据库迁移 | SQLite schema 变更脚本 |
| `dev/` | 开发辅助 | 开发环境搭建、数据生成等 |
| `build/` | 构建辅助 | 打包前后的额外处理 |

## 建议添加的脚本

| 脚本 | 用途 |
|------|------|
| `dev/setup.ps1` | 一键搭建开发环境 |
| `dev/seed-data.ts` | 生成测试用的剪贴板记录 |
| `migrate/v1.sql` | 数据库初始迁移 |
