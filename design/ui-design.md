# History Clipboard — 界面设计

## 1. 全局设计语言

### 1.1 设计原则

- **清晰**：信息层级分明，一眼定位目标内容
- **快速**：核心操作不超过 2 步（唤起 → 选择 → 完成）
- **低调**：后台静默运行，浮窗用时出现、不用时消失

## 2. 浮窗（QuickPick）设计

### 2.1 布局结构

```
┌──────────────────────────────────────┐
│ 🔍 搜索历史...                   ⚙️  │  ← 搜索栏 (40px)
├──────────────────────────────────────┤
│ 📌 这是置顶的第一条内容预览...   ⭐📌 │  ← 置顶项 (金色左边框)
│    3 分钟前 · 来自 Chrome           │
├──────────────────────────────────────┤
│ 这是收藏的重要参考文本内容...    ⭐  │  ← 收藏项 (星标高亮)
│    1 小时前 · 来自 VS Code          │
├──────────────────────────────────────┤
│ 普通的历史剪贴板内容第二行...        │  ← 普通项
│    刚刚 · 来自 Notion               │
├──────────────────────────────────────┤
│ 又一条普通的历史记录内容...          │  ← 普通项
│    2 分钟前 · 来自 WeChat           │
├──────────────────────────────────────┤
│ ...更多条目（滚动）                  │
├──────────────────────────────────────┤
│ 共 247 条历史 · 最近 3 天           │  ← 底部状态栏 (32px)
└──────────────────────────────────────┘
```

### 2.2 条目设计

每个条目高度自适应（最小 48px），包含：

```
┌──────────────────────────────────────────┐
│ [图标]  内容预览文字（最多两行截断）...  │  ← 主要区域
│         2 分钟前 · Chrome                 │  ← 元信息
│                                    [⭐] [📌] │  ← 快捷操作（hover 显示）
└──────────────────────────────────────────┘
```

- **内容预览**：单行最大 60 字符，超出显示省略号；长文本自动折行（最多 2 行）
- **类型图标**：文本=T 图标、图片=🖼️ 图标
- **时间显示**：刚刚 / N 分钟前 / N 小时前 / N 天前 / 具体日期
- **来源应用**：显示窗口标题（截取应用名）
- **收藏/置顶按钮**：鼠标悬停时浮现，已激活状态常驻显示

### 2.3 交互行为

| 操作 | 行为 |
|------|------|
| 点击条目主体 | 复制内容到系统剪贴板 → 浮窗隐藏 → 系统通知「已复制」 |
| 点击 ⭐ | 切换收藏状态，即时更新样式 |
| 点击 📌 | 切换置顶状态，列表重新排序 |
| 顶部搜索框输入 | 实时过滤，高亮匹配文字 |
| 按 ↑↓ 键 | 在列表中上下移动选择焦点（蓝色边框高亮） |
| 按 Enter | 复制当前高亮条目并隐藏 |
| 按 Esc / 失焦 | 隐藏浮窗（不复制） |
| 拖拽窗口边缘 | 调整窗口大小 |

### 2.4 键盘导航

```
Esc           → 隐藏浮窗
↑ / ↓         → 上下移动焦点
Enter         → 复制选中条目 + 隐藏
Ctrl+1~9      → 快速复制第 1~9 条
Backspace     → 清空搜索框
```

### 3.2 左侧列表面板

- **分组显示**：置顶 → 收藏 → 历史记录，每组有折叠/展开箭头
- **条目高度**：56px 固定高度
- **选中高亮**：蓝色左边框 + 浅蓝背景
- **右键菜单**：复制 / 收藏(取消) / 置顶(取消) / 编辑 / 删除
- **多选模式**：Ctrl+点击 多选，Shift+点击 范围选择，顶部出现批量操作栏

### 3.3 右侧详情面板

- 显示选中条目的**完整内容**（可滚动）
- 文本内容使用等宽字体、保留换行和缩进
- 元信息：来源应用、创建时间、最后使用时间、内容字数/大小
- 底部操作按钮组：[复制到剪贴板] [⭐ 收藏] [📌 置顶] [🗑 删除]
- 未选中任何条目时显示：应用 Logo + 统计概览

### 3.4 工具栏

| 元素 | 功能 |
|------|------|
| 搜索框 | 全文模糊搜索，实时过滤列表 |
| 类型筛选 | 下拉选择：全部 / 文本 / 图片 |
| 收藏筛选 | 下拉：全部 / 仅收藏 / 仅置顶 |
| [全部清除] 按钮 | 弹出确认对话框后清除非收藏+非置顶内容 |

### 4.2 各设置项交互

| 设置项 | 控件类型 | 行为 |
|--------|----------|------|
| 主题 | Radio 按钮组 | 即时切换 |
| 强调色 | 色块按钮组 | 点击即时预览 |
| 字体大小 | Radio 按钮组 | 即时切换 |
| 开机自启 | Switch 开关 | 即时生效 |
| 快捷键 | 快捷键捕获输入框 | 点击后按下新快捷键组合，冲突时红色提示 |
| 保留天数 | Select 下拉 | 选择即生效 |
| 最大记录数 | InputNumber | 失焦或回车生效 |

### 4.3 设置存储

所有设置项通过 IPC `settings:set` 实时写入 SQLite settings 表，同时更新主进程相关模块（如快捷键变更需重新注册）。

## 5. 交互动效

| 场景 | 动效 | 时长 |
|------|------|------|
| 浮窗出现 | 从鼠标位置轻微放大 + 淡入 (scale 0.9→1, opacity 0→1) | 150ms |
| 浮窗隐藏 | 淡出 (opacity 1→0) | 100ms |
| 条目悬停 | 背景色平滑过渡 | 150ms |
| 收藏/置顶 | 星标/图钉图标弹跳动画 | 300ms |
| 列表刷新 | 新条目从顶部滑入 | 200ms |
| 删除 | 条目向右滑出 + 高度收缩 | 200ms |
| 复制成功 | 条目短暂绿色闪烁 | 400ms |
| 主题切换 | 全局颜色平滑过渡 | 300ms |

## 6. 系统通知

| 场景 | 通知内容 | 方式 |
|------|----------|------|
| 复制成功 | 不弹通知（避免干扰） | — |
| 自动清理完成 | 「已自动清理 N 条过期记录」 | Windows 原生通知（仅当 N > 10） |
| 首次启动 | 「History Clipboard 已在后台运行，Ctrl+Shift+V 唤起」 | Windows 原生通知 |
| 快捷键冲突 | 「快捷键已被占用，请在设置中更换」 | Windows 原生通知 |

## 7. 应用图标设计方向

- **主图形**：剪贴板/便签纸造型，叠加时钟元素暗示「历史」
- **配色**：与默认强调色一致（`#1677FF` 蓝）
- **尺寸**：提供 16/32/48/64/128/256 px（.ico 格式）
- **托盘图标**：简洁单色版本，确保在 Windows 任务栏 16×16 下可辨识
