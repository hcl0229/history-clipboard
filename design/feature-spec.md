# History Clipboard — 功能规格

## 文档约定

| 标记 | 含义 |
|------|------|
| P0 | 核心功能，MVP 必须实现 |
| P1 | 重要功能，首个正式版应包含 |
| P2 | 增强功能，后续迭代加入 |
| ⚠️ | 边界条件需特别关注 |

---

## 模块一：剪贴板监控与存储

### F1.1 内容类型支持

| ID | 功能 | 优先级 | 描述 |
|----|------|--------|------|
| F1.1.1 | 纯文本 | P0 | 所有纯文本内容，无格式 |
| F1.1.2 | 富文本（HTML） | P1 | 带格式文本，存储为 HTML 字符串 |
| F1.1.3 | 图片 | P1 | 存储为 Base64 或文件路径引用 |

### F1.2 监控机制

| ID | 功能 | 优先级 | 描述 |
|----|------|--------|------|
| F1.2.1 | 定时轮询 | P0 | 每 300ms 读取系统剪贴板，检测变化 |
| F1.2.2 | 哈希去重 | P0 | 内容 SHA256 哈希，与最近一条对比，相同则跳过 |
| F1.2.3 | 来源记录 | P1 | 获取当前前台窗口标题，记录内容来源 |

**边界条件** ⚠️：
- 应用启动时不清空系统剪贴板
- 用户手动复制已存在的历史内容 → 不重复记录（哈希去重）
- 图片过大（>10MB）→ 跳过，写入日志
- 剪贴板为空/读取失败 → 静默跳过本轮

### F1.3 自动清理

| ID | 功能 | 优先级 | 描述 |
|----|------|--------|------|
| F1.3.1 | 按天清理 | P0 | 非收藏+非置顶内容超过配置天数后自动删除 |
| F1.3.2 | 按量清理 | P1 | 总记录数超出上限时，删最旧的 非收藏+非置顶 记录 |
| F1.3.3 | 清理时机 | P0 | 应用启动时执行一次 + 之后每 1 小时执行一次 |

**配置项**：
- 保留天数：默认 3 天，可选 1 / 3 / 7 / 14 / 30 天
- 最大记录数：默认 1000，可选 100 ~ 10000

### F1.4 手动清理

| ID | 功能 | 优先级 | 描述 |
|----|------|--------|------|
| F1.4.1 | 全部清除 | P0 | 仅删除 非收藏+非置顶 内容，清除前弹出确认对话框 |
| F1.4.2 | 单条删除 | P0 | 右键菜单或删除按钮，任何内容均可删除 |
| F1.4.3 | 批量删除 | P1 | 多选后批量删除 |

**确认对话框文案** ⚠️：
> 确定要清除全部历史记录吗？（收藏和置顶内容不受影响）
> 共 N 条记录将被删除，此操作不可撤销。

## 模块二：收藏与置顶

### F2.1 收藏（Favorite）

| ID | 功能 | 优先级 | 描述 |
|----|------|--------|------|
| F2.1.1 | 添加收藏 | P0 | 对任意一条历史记录标记收藏 |
| F2.1.2 | 取消收藏 | P0 | 取消标记，恢复为普通项（受清理规则影响） |
| F2.1.3 | 收藏筛选 | P1 | 主窗口中可按「仅收藏」筛选列表 |

**收藏规则**：
- 收藏内容永久保留，不受天数清理和容量清理影响
- 收藏内容不受「全部清除」影响
- 收藏内容仍可被手动单条删除

### F2.2 置顶（Pin）

| ID | 功能 | 优先级 | 描述 |
|----|------|--------|------|
| F2.2.1 | 置顶 | P0 | 将内容固定到列表最顶部 |
| F2.2.2 | 取消置顶 | P0 | 取消置顶，恢复正常排序位置 |
| F2.2.3 | 多条置顶 | P0 | 支持多条同时置顶 |

**置顶规则**：
- 置顶组始终排在列表最前面
- 置顶组内部按创建时间倒序排列
- 置顶内容不受「全部清除」影响
- 置顶内容不受天数清理影响（但在后续版本中可能调整）

### F2.3 排序规则

```
排序优先级：置顶 → 收藏 → 时间

  置顶项（按时间倒序）
  ─────────────────────
  收藏项（按时间倒序）
  ─────────────────────
  普通项（按时间倒序）
```

## 完整交互流程

### 日常使用流程

```
用户复制内容 (Ctrl+C)
    → 后台自动记录
    ↓
用户需要粘贴历史内容
    → 按 Ctrl+Shift+V
    → 浮窗弹出，显示最近 30 条
    → 上下键快速浏览 / 输入关键词搜索
    → Enter 或点击选中
    → 内容复制到系统剪贴板
    → 浮窗自动隐藏
    → 用户在目标应用 Ctrl+V 粘贴
```

### 管理流程

```
双击托盘图标 / 右键→打开主窗口
    → 主窗口打开
    → 浏览全部历史
    → 搜索/筛选定位内容
    → 对重要内容：右键→收藏 / 置顶
    → 对不需要的内容：右键→删除
    → 定期：设置→全部清除（非收藏/非置顶）
    → 关闭窗口→最小化到托盘（后台继续监控）
```

### 清理流程

```
[自动] 应用启动时 / 每1小时
    → 计算截止时间 = now - 3天
    → DELETE WHERE created_at < 截止时间 AND NOT favorite AND NOT pinned
    → 检查总数 > 1000?
        → DELETE 最旧的非收藏非置顶记录至 ≤1000

[手动] 用户点击「全部清除」
    → 弹出确认对话框
    → 用户确认
    → DELETE WHERE NOT favorite AND NOT pinned
    → 提示 "已清除 N 条记录"
```

## 非功能需求

| 维度 | 要求 |
|------|------|
| 性能 | 浮窗唤起 < 300ms，列表滚动 60fps |
| 内存 | 后台常驻 < 150MB |
| 存储 | 千条记录数据库 < 10MB |
| 兼容性 | Windows 10 / 11（x64） |
| 安装包 | < 100MB（Electron 典型体积） |
| 更新 | 支持自动检查更新（P2） |
