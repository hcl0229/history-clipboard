# History Clipboard — 界面设计 v2

> 版本：v2.2 | 日期：2026-06-11
> 状态：QuickPick 重构完成 ✅（v2.1 双向修复 B9+B11）

---

## 1. 设计原则

- **轻量**：窗口足够小（QuickPick 约 340×420），不遮挡工作区
- **快速**：Ctrl+Shift+V 唤起 → ↑↓ 选择 → Enter 粘贴，全程 <2 秒
- **低调**：关闭即隐藏到托盘，不退出

## 2. 色彩系统（已实现）

| 用途 | 色值 |
|------|------|
| 置顶背景 | `rgba(255,77,79,0.07)` 淡红 |
| 收藏背景 | `rgba(250,173,20,0.08)` 浅黄 |
| 置顶图标 | `#ff4d4f` |
| 收藏图标 | `#faad14` |
| 未激活图标 | `#999` |

## 3. QuickPick 浮窗

### 3.1 尺寸

- 宽度：max(340, 屏幕宽 × 18%)
- 高度：max(420, 屏幕高 × 40%)
- 最小：280 × 280
- 无边框 (`frame: false`)，毛玻璃效果
- 标题栏区域可拖拽 (`-webkit-app-region: drag`)

### 3.2 布局

```
┌────────────────────────────────┐
│ 🔍 搜索...                     │ ← 标题栏（可拖拽）
│ 12 条 · ↑↓ Enter Esc       ↻  │ ← 统计 + 刷新
├────────────────────────────────┤
│ 第一条内容预览...      ⭐ 📌   │ ← 置顶项（淡红背景）
│ 刚刚                          │
│ 第二条内容预览...      ☆ 📌   │ ← 收藏项（浅黄背景）
│ 3分前                         │
│ 第三条内容预览...      ☆ 📍   │ ← 普通项
│ 1时前                         │
│ ...（可滚动）                  │
└────────────────────────────────┘
```

### 3.3 关键设计决策（v2 重构）

**❌ 废弃**：Ant Design `<List>` + `<Paragraph>` 渲染列表项。
**原因**：AD 组件内部事件机制与自定义 onClick 冲突，经过 5 轮修复仍无法正常触发星标/图钉点击。

**✅ 新方案（v2.0）**：用原生 `<div>` + CSS 替换。
**✅ v2.1 修复**：`useState` 本地渲染状态替代直接 `useClipboardStore.setState()`，保证 React 重渲染。

- 列表：`<div>` + `.map()` 代替 `<List>`
- 文本截断：CSS `text-overflow: ellipsis` 代替 `<Paragraph ellipsis>`
- 点击分离：`.item-content onClick` → 复制，`.item-actions span onClick` → 收藏/置顶
- **渲染源**：React `useState(renderItems)` 作为渲染源，乐观更新直接走 `setRenderItems(prev => ...)` 保证 React 100% 重渲染
- **拖拽**：顶部 14px `.quickpick-drag-handle` 专用于窗口拖拽

### 3.4 交互行为

| 操作 | 行为 |
|------|------|
| 点击条目正文 | 复制到剪贴板 → 隐藏窗口 |
| 点击 ⭐ | 乐观更新切换收藏 → 广播 `clipboard:itemUpdated` |
| 点击 📌 | 乐观更新切换置顶 → 广播 `clipboard:itemUpdated` |
| Ctrl+V | 全局快捷键唤起/关闭浮窗 |
| ↑↓ / Enter / Esc | 键盘导航 |

## 4. 主窗口

### 4.1 尺寸

- 宽度：max(680, 屏幕宽 × 70%)
- 高度：max(500, 屏幕高 × 72%)
- 最小：500 × 380

### 4.2 布局（已实现）

```
┌──────────────┬──────────────────┐
│ 🔍 搜索...   │                  │
│ [剪切板][收藏]│   内容详情       │
├──────────────┤                  │
│ 条目1  ⭐📌  │  完整文本内容    │
│ 条目2  ☆📍  │                  │
│ 条目3  ☆📍  │  来源 | 时间     │
│ ...          │                  │
│ (可滚动)     │  [⭐][📌][📋][🗑]│
├──────────────┤                  │
│ [清空] [↻]   │                  │
└──────────────┴──────────────────┘
```

### 4.3 关键设计决策（已实现）

- **2 Tab**：剪切板（置顶项排在最上）| 收藏
- **无折叠分组**：所有项在单一排序列表中
- **每行始终显示 ⭐📌 图标**：激活=彩色，未激活=灰色
- **图标可点击**：直接点列表中的图标即可收藏/置顶
- **左侧用原生 div**：不用 Ant Design `Layout.Sider`（避免 overflow hidden）
- **统计数据实时从 items 派生**：不用独立计数器

## 5. 设置面板

- 最大宽度 380px，居中
- 垂直表单布局，`size="small"`
- 控件：主题 Select、强调色 Select、字号 Slider、保留天数 Slider、最大记录 Slider、开机自启 Switch

## 6. 拖拽窗口

QuickPick 窗口 `frame: false`，标题区域使用 CSS `-webkit-app-region: drag`，输入框和按钮使用 `no-drag`。
