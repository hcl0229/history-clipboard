# data/ — 开发数据

> 版本：v1.0 | 日期：2026-06-09

此目录存放**开发和测试阶段**的静态数据，不包含运行时数据。

## 目录约定

| 子目录 | 用途 | 说明 |
|--------|------|------|
| `fixtures/` | 测试夹具 | 单元测试和 E2E 测试用的样本剪贴板内容 |
| `seed/` | 种子数据 | 开发环境初始化用的预设剪贴板记录 |
| `templates/` | 数据模板 | 批量导入或配置文件模板 |

## 运行时数据

应用运行时数据存储在用户 AppData 目录，**不在此目录中**：

```
%APPDATA%/history-clipboard/
├── clipboard.db          # SQLite 数据库
├── settings.json         # 运行时设置缓存
└── logs/                 # 日志文件
```
